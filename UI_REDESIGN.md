# Skills House - 清新现代界面重构

## 🎨 设计理念

### 美学方向：清新自然 + 现代极简
- **灵感来源**：春日微风、有机形状、柔和渐变
- **核心感受**：轻盈、舒适、精致
- **差异化**：流动的背景、细腻的动画、独特的字体搭配

---

## ✨ 新设计特性

### 1. 🎨 **色彩系统**

#### 主色调（清新自然）
- **春绿色** (`#A8E6CF`) - 生机与活力
- **薄荷绿** (`#DCF8E8`) - 清新与舒适
- **鼠尾草浅色** (`#F0F7F4`) - 背景基调
- **平静海洋** (`#6DD3CE`) - 强调色
- **柔和珊瑚** (`#FFB6A3`) - 提示/警告
- **薰衣草雾** (`#E8D5F2`) - 装饰点缀

#### 渐变
```css
--gradient-spring: linear-gradient(135deg, #A8E6CF 0%, #DCF8E8 50%, #E8D5F2 100%);
```

### 2. ✍️ **字体系统**

#### 主字体家族
- **Outfit** - 标题和Logo（现代几何感）
- **DM Sans** - 正文（易读性强）

避免了常见的 Inter、Roboto、Arial 等通用字体

### 3. 🌊 **有机背景**

#### 动态浮动装饰
- 两个大型径向渐变圆形
- 20-25秒循环动画
- 轻微旋转和位移
- 营造深度和氛围

```css
animation: float 20s ease-in-out infinite;
```

### 4. 🎬 **动画系统**

#### 页面加载动画
- **Header**: 下滑淡入 (0.6s)
- **搜索栏**: 淡入上滑 (0.8s, delay 0.2s)
- **Skills 卡片**: 淡入上滑 (1s, delay 0.4s)
- **每张卡片**: 递增延迟 (0.1s/张)

#### 交互动画
- **悬停**: 上浮 + 阴影增强
- **点击**: 轻微缩放反弹
- **输入框焦点**: 上浮 + 边框发光
- **卡片顶部**: 渐变条展开

#### 缓动函数
```css
cubic-bezier(0.16, 1, 0.3, 1)  /* 主要用于页面加载 */
cubic-bezier(0.34, 1.56, 0.64, 1)  /* 用于弹性交互 */
```

### 5. 📐 **空间设计**

#### 布局特点
- **慷慨的留白** - 视觉呼吸空间
- **圆角设计** - 24-32px 大圆角
- **柔和阴影** - 低对比度、大范围
- **模糊背景** - backdrop-filter 毛玻璃效果

#### 卡片设计
- 渐变顶部条（hover 展开）
- 半透明白色背景
- 多层阴影叠加
- 边框使用主题色

### 6. 🎯 **细节优化**

#### 微交互
- Logo 文字渐变色
- 用户名标签背景色
- 搜索图标定位
- 版本号徽章
- 按钮渐变背景

#### 视觉层次
- 毛玻璃效果（backdrop-filter）
- 多层阴影
- 颜色透明度变化
- Z-index 精确控制
